# **üìå Variables & Constants in PL/SQL**

---

### **1Ô∏è‚É£ Variables**

A **variable** is a named memory location to store a value that can **change** during execution.

#### **Syntax**

```sql
variable_name datatype [NOT NULL] [:= initial_value];
```

* `datatype` ‚Üí Any Oracle datatype (`NUMBER`, `VARCHAR2`, `DATE`, etc.)
* `NOT NULL` ‚Üí Means the variable must always have a value.
* `:=` ‚Üí Assignment operator for initialization.

---

#### **Example ‚Äì Declaring and Using a Variable**

```sql
SET SERVEROUTPUT ON;

DECLARE
    v_student_name VARCHAR2(50);
    v_marks NUMBER(5,2) := 87.50;
BEGIN
    v_student_name := 'Akshit Patel';
    DBMS_OUTPUT.PUT_LINE('Name: ' || v_student_name);
    DBMS_OUTPUT.PUT_LINE('Marks: ' || v_marks);
END;
/
```

---

### **2Ô∏è‚É£ Constants**

A **constant** is like a variable, but its value **cannot be changed** once assigned.

#### **Syntax**

```sql
constant_name CONSTANT datatype := value;
```

---

#### **Example ‚Äì Constant**

```sql
SET SERVEROUTPUT ON;

DECLARE
    c_pi CONSTANT NUMBER := 3.14159;
    v_radius NUMBER := 5;
    v_area NUMBER;
BEGIN
    v_area := c_pi * v_radius * v_radius;
    DBMS_OUTPUT.PUT_LINE('Area: ' || v_area);
END;
/
```

üí° If you try to change `c_pi` later in the block, Oracle will throw an error.

### **3Ô∏è‚É£ Using `%TYPE`**

* Lets you declare a variable with the **same datatype as a table column**.
* Avoids hardcoding datatypes ‚Üí safer if table changes.

#### Example

```sql
DECLARE
    v_name students.name%TYPE;
BEGIN
    SELECT name INTO v_name
    FROM students
    WHERE student_id = 1;

    DBMS_OUTPUT.PUT_LINE('Name: ' || v_name);
END;
/
```

---

### **4Ô∏è‚É£ Using `%ROWTYPE`**

* Declares a variable that can hold **an entire row** from a table.
* Useful when you want all column values at once.

#### Example

```sql
DECLARE
    v_student students%ROWTYPE;
BEGIN
    SELECT * INTO v_student
    FROM students
    WHERE student_id = 1;

    DBMS_OUTPUT.PUT_LINE('Student: ' || v_student.name || ', Email: ' || v_student.email);
END;
/
```

---

‚úÖ **Key Points to Remember**

* **Variables** ‚Üí can change value
* **Constants** ‚Üí fixed value, must initialize at declaration
* `%TYPE` ‚Üí matches a column‚Äôs datatype
* `%ROWTYPE` ‚Üí matches a table‚Äôs row structure
* Always `SET SERVEROUTPUT ON` when using `DBMS_OUTPUT.PUT_LINE`
